## 🟡 有理數

### 1️⃣️ 定義

> **📌 理論:**
>
> 💡已知: 兩整數 m 、 n, n ≠ 0;
>
> 若: 存在一數 a = m / n
>
> 則: 稱 a 為有理數 (其中: 若 m 、 n 互質時，則稱為最簡分數)

### 2️⃣ 有理數與小數的關係

> **📌 理論:**
>
> 💡已知: a 為一個有理數 (a = m / n; m, n ∈ ℤ, n ≠ 0;)
>
> 若: 將 a 化成十進位的小數
>
> 則: a 必為
>
> 1. 整數: n 為 m 的因數，m 為 n 的倍數

> - 分母異於 2, 5 次方的因數必為分子的因數

> 3.  無窮循環小數

## 🟡 實數

### 1️⃣️ 定義

> **📌 理論:**
>
> 💡已知一數線
>
> 若: 數線上每一個點都對應為一個數值
>
> 則: 稱這些所有的數值為實數
>
> 即: 數線上找得到的數就是實數
>
> 其中實數可以分成: 1. 有理數, 2. 無理數

### 2️⃣ 無理數

> **📌 理論:**
>
> 💡已知一實數 χ
>
> 若: χ 無法 化成 n/m; m,n ∈ ℤ; m ≠ 0 的表示法
>
> 則: χ 為無理數
>
> 其中: χ 必為無限不循環小數
>
> 比如: 常見的無理數有 √￣ , π, log, sinΘ

### 3️⃣ 無理數與有理數計算

- 有理數 + 有理數: 有理數
- 有理數 x 有理數: 有理數
- 有理數 + 無理數: 無理數
- 有理數 x 無理數: 無理數 or 0 (π x 0)
- 無理數 + 無理數: 實數 (π+(-π))
- 無理數 + 無理數: 非零實數 (π x 1/π)

```note
If a, b, c, d ∈ ℚ
a + b√2 = c + d√2
a - c = (b - d)√2
a - c = b - d = 0
```

> 1. ℚ(+-×÷)ℚ = ℚ

### 4️⃣ 反證法(歸謬法)

> **📌 理論:**
>
> 💡反證法就是從**結論的反面**出發，透過一系列正確無誤的推理，最後導致與現實條件、公設、定義、公理...等等數學上已知的事實相矛盾，而得出**結論的反面**不成立，從而肯定**命題的結論**是正確的。

> 【反證法的步驟】
>
> Step1：反設命題：否定命題中的結論。
>
> Step2：導出矛盾：把反面的假設作為輔助條件，添加到命題的前提中，從這些條件出發，最後導出矛盾。
>
> Step3：肯定結論：**否定命題的結論**不成立，所以肯定**命題的結論**

> **📌 範例:**
>
> (1) 試證 √5 為無理數。
>
> (2) 由上題之結果，試證 √5 + 3 為無理數。
>
> (3) 承(1)(2) 題，試證 √5 + ∛2 為無理數。

```Proof
(1) 反設 √5 為有理數，即 √5 = p/q (其中 p, q ∈ ℤ，p ≠ 0，(p, q) = 1) ......①
    平方 ⇒ 5 = p²/q² ⇒ 5 | q² ⇒ 5 | q ......②
    可設 q = 5k，k ∈ ℤ，代入 5p² = q² 中，得 5p² = (5k)²
    ⇒ p² = 5k² ⇒ 5 | p² ⇒ 5 | p ......③
    由①②③知矛盾，故 √5 為無理數
(2) 反設 √5 + 3 為有理數，即設 √5 + 3 = a ∈ ℚ，
    ⇒ √5 = a - 3 ∈ ℚ，與(1)之結論不合，故矛盾 ∴√5 + 3 為無理數
(3) 反設 √5 + ∛2 為有理數，即設 √5 + ∛2 = b ∈ ℚ，
    ⇒ ∛2 = b - √5 兩邊立方得 2 = b³ - 3√5b² + 15b - 5√5
    ⇒ √5 = (b³ + 15b - 2) / 3b² + 5 ∈ Q，與(1)之結論不合，故矛盾
    ⇒ √5 + ∛2 為無理數
```

### 7️⃣ 共軛無理數

> **📌 理論:**
>
> 💡 已知： a, b, c ∈ ℚ; 且 √c ∉ ℚ
>
> 則: 稱 a + b√c 與 a - b√c 互為共軛無理數
>
> 其中:
>
> 1. (a + b√c) + (a - b√c) = 2a
> 2. (a + b√c) × (a - b√c) = a² + b²c
>    💡 已知： a, b, p, q ∈ ℚ; 且 √c ∉ ℚ
>
> 若: a + b√c = p + q√c
>
> 則: a = p, b = q

```proof
let b ≠ q
⇒ a + b√c = p + q√c
⇒ (b - q)√c = p - a
⇒ √c = (p - a) / (b - q) ∈ ℚ 與已知不合
Hence, b = q ⇒ p - a = 0 ⇒ a = p
```
