# 📗 國中數學筆記

這裡是 **國中數學** 筆記，涵蓋 **國一到國三** 的所有數學知識，包括 **代數、幾何、函數、統計與機率** 等主題。

## 🔢 內容大綱

### 🎯 國一數學

- **整數與有理數**：四則運算、絕對值
- **方程式**：一次方程式、二元一次方程式
- **比與比例**：比例式、百分比應用
- **平面幾何**：三角形、四邊形

### 📊 國二數學

- **因式分解**：多項式運算
- **函數與圖形**：座標平面、一次函數
- **統計與機率**：機率計算、長條圖、圓餅圖
- **三角比**：正弦、餘弦、正切

### 🏆 國三數學

- **二次函數**：拋物線、最大最小值
- **三角函數**：特殊角度計算
- **圓的性質**：切線、弧長、圓周角
- **排列與組合**：組合數、概率應用
