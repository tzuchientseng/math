### 2️⃣ 印象深刻的題

💡 開課題: 一正整數的兩最小因數和為 6 最大的兩因數的和為 462 則此正整數為，則此正整數為? (線段和倍問題)

```Sol
因數: 1, 5 ... □, ○
□ = 1 x □ = 5 x ○
462 ÷ (5 + 1) = 77
77 x 5 = 385
462 - 77 = 385; // (77*5)
```
