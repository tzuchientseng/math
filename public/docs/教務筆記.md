# 📜 教務筆記

## ⚠️ 重要注意事項

- 字體大小: 國中講義14(Mac12), 考卷12; 字體標楷體，數字timesNewRoman
- 段落: 位移點數: 0.8cm;行距多行 1.25;
- 要插入分頁符號
- 版本: S版, 專班(e.g. 石牌薇閣, 復興專班), A版
- MathType: 記得format or 格式化

## ‼️ 其他注意事項

- 圖不要切到其他題目
- a,b 改成 _A_、_B_(斜體)
- =\_\_\_\_(等號後不用空格)
- 角度用(指數＋∘)
- 題目間可留空行（空行請用 Shift + Enter），選擇題可留 1 行即可，填充題可多留一些
- 避免頁與頁之間跑掉，使用分頁符號（版面配置 → 分隔符號 → 分頁符號）
- 數字（type）大小調整為 12（先進其中一個 type，從 size → define → 最上面的字體大小調整為 12，之後其他的可用內建按鈕 “format” 全部格式化）
- 建檔完跟原檔案做比對（檢查），若有少字的部分要增加，有錯誤的部分進行調整
- 人名記得加私名號（國家、地標…等也是）
- 有時候題目沒有標點符號 or 其他空格類，需要自行判斷增減
- 若是考卷上沒有分題型的話，那也不需要分，代表是混合題型（#22 為選擇，夾在填充的中間）
